# Personal Investment Portfolio Optimization

## Overview
This project applies Modern Portfolio Theory (MPT) to optimize an investment portfolio using a mix of ETFs. It analyzes historical market data, generates the Efficient Frontier, identifies optimal portfolios, and evaluates performance through backtesting.

## Features
- Historical ETF data analysis
- Portfolio optimization using MPT
- Efficient Frontier visualization
- Maximum Sharpe Ratio and Minimum Variance portfolios
- Portfolio backtesting
- Tax-aware asset allocation

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- yfinance

## Dataset
ETF price data collected from Yahoo Finance for:
- SPY
- AGG
- VEA
- VWO
- GLD
- IYR
